Генератор файла для импорта тегов в OPC UA

Инструкция по использованию

Как пользоваться генератором:

  1. Решения: Загрузите XML файл с решением через кнопку "Добавить". Выберите решение из списка для работы.
  2. Тип объекта: Выберите тип объекта из списка всех типов, найденных в решении.
  3. Переключение между сопоставлением и объектами: Используйте вкладки для переключения между таблицами сопоставления и списком объектов.
  4. Сопоставление: Можно загрузить из CSV или сохранить на сервер.
  5. Объекты: Используйте кнопку "Выгрузить данные из решения" для загрузки объектов выбранного типа. Также можно загрузить свой список из CSV файла.
  6. Управление чеками: Используйте кнопки "Убрать чеки"/"Поставить чеки" для быстрого управления всеми чекбоксами, или "Поставить чеки только где есть Головной тег" для выборочной установки.
  7. Генерация: Выберите нужные опции и нажмите "Сгенерировать" для создания файла импорта.
